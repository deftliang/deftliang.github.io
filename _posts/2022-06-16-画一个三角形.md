# 画一个三角形

今天我们来画一个常用的图形(三角形)吧!

用 css 方式可以这样:

```css
.triangle {
  width: 0;
  height: 0;
  border-bottom: 10px solid red;
  border-left: 10px solid transparent;
  border-right: 10px solid transparent;
}
```

用 SVG 方式可以这样写:

```XML
<svg width="100" height="100">
  <path d="M0,100 100, "></path>
</svg>
```
